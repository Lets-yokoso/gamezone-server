# GameZone Server

## Setup
```bash
npm install
npm start
```

## Environment Variables
- `PORT` — default 3000
- `JWT_SECRET` — change this for production!

## Hosting Options
1. **Local (LAN)** — Run on any PC. Use your local IP (e.g. 192.168.1.100:3000)
2. **Cloud** — Deploy to Railway, Render, or any Node.js host for remote access
